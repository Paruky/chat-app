# Paruky Chat

## 概要
友達用のPWAチャットアプリ

## 技術構成
- Frontend: HTML/CSS/JS
- Backend: Supabase
- Hosting: Render

## 作る上での注意
- モバイルファースト
- iPhone SE など画面が小さいモバイルでも見やすいように
- わかりやすいUI
- 動作は軽く
- バグは少なく
- とにかく安定した

## 主な機能
- リアルタイムチャット
- PWA
- タイピング表示

## 現在の問題・バグ
- iOS PWAで送信ボタンが反応しないことがある
- 部屋に入るたびに部屋のリスト順が変わってしまう

## 今後やりたいこと(上からやりたい順)
- UIの整理
- メッセージ削除
- 返信機能
- 画像送信
- URL自動リンク

## 開発環境起動

- npm install
- npm run dev

## 使用サービス

- Supabase realtime
- Render deploy
